# 2Dアクションゲーム「Mono-Jump」実装プラン

角が丸い一つ目のキャラクターが全15ステージを攻略するアクションゲームです。壁登りアクションと滑らかな移動を特徴とします。

## 提案される変更

### 1. 新規プロジェクト作成: `mono_jump/`
- **[NEW] `index.html`**: ゲーム画面、UIレイアウト、モバイル操作用パッドの定義。
- **[NEW] `style.css`**: ゲームデザイン（ダークモード基調）、キャラクターのスタイリング（Canvas描画補助）、アニメーション。
### 4. 壁アクションのスクラッチ風改造
- **[MODIFY] [script.js](file:///Users/pc13admin/Documents/kikennmodsett/mono_jump/script.js)**
    - **壁ずり（Wall Slide）**: 壁に接触しながら落下している際、落下速度を一定値に制限し、壁に張り付いている感覚を強化します。
    - **壁判定の強化**: 速度が0の状態でも方向キーを壁側に押していれば壁ジャンプが可能な状態を維持します。
    - **ジャンプ挙動の調整**: ジャンプ時の初速と横移動の減衰を調整し、動画で見られる「キビキビとした壁登り」を再現します。
    - **入力受付の改善**: 壁ジャンプの先行入力や、壁から離れた直後の猶予フレーム（コヨーテタイムの壁版）の検討。

### 2. キャラクターデザイン
- Canvasの `roundRect` と `arc` を使用して、角丸四角形と大きな一つ目を描画。

## 検証プラン

### 自動テスト
- ステージデータの整合性チェック。

### 手動検証
- **物理挙動**: 壁に向かってジャンプした際の挙動がスムーズか。
- **操作性**: キーボードとモバイル（仮想パッド）で問題なく操作できるか。
- **ギミック**: 棘に触れた際にリスタートするか、ゴールで次のステージに進むか。
- **タイマー**: ゲーム開始から15ステージクリアまでタイムが正確に計測・表示されるか。
